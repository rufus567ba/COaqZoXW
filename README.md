# 项目简介

## 核心功能
- 基于 YOLOv8 的飞机安检图像检测系统
- 支持 4 类物品检测：电池(battery)、液体(liquid)、机器(machine)、金属电池机器液体(metal-battery-machine-liquid)
- 提供完整源码、数据集、训练好的模型及训练结果
- 支持图片、视频及摄像头进行实时检测
- 支持批量检测并查看所有结果
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和 IOU 阈值
- 支持选择显示标签和原图
- 支持选择不同检测模型
- 提供完整的项目运行教程

## 适用场景
- 机场安检图像自动检测
- 安检设备辅助识别系统
- 安防监控图像分析
- 安全检查培训系统

## 技术特点
- 采用 Pycharm+Anaconda 或 VSCode+Anaconda 开发环境
- 基于 Python 3.9，依赖 opencv-python、PyQt5、ultralytics、torch1.9 等库
- 提供 UI 界面，操作便捷
- 内置 180 张标注数据集
- 已训练完成模型，配置环境后可直接使用
- 支持多输入源：单张图片、图片文件夹、视频文件、本地摄像头
